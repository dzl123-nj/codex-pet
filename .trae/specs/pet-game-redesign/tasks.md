- [x] Task 9: 性能优化
  - 使用 v-show 替代 v-if 切换频繁显示的元素（卡牌、地鼠格子）
  - 卡牌翻转使用 CSS transform + will-change 优化
  - 地鼠动画使用 CSS animation 而非 JS 定时器驱动
  - 所有动画使用 transform 和 opacity（GPU 加速），避免触发 layout/paint
  - 使用 requestAnimationFrame 替代 setInterval 进行倒计时
  - 粒子效果使用 CSS 实现，不引入额外库
  - 卡牌图案使用 emoji 而非图片，减少资源加载
  - localStorage 读写使用 try-catch 防止异常
  - 道具定时恢复使用时间戳比较，而非持续运行定时器
  - 所有定时器 ID 存储在 data 中，游戏结束/对话框关闭时清除
  - 使用 this.$once('hook:beforeDestroy', ...) 确保定时器清理
  - 统一游戏区域配色方案
  - 添加页面切换过渡动画（菜单→关卡选择→游戏→结果）
  - 确保响应式布局正常
